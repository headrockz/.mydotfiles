# Asafe's dotfiles

This repository holds my dotfiles and contains a python script to backup them and commit

I use [oh-my-zsh](https://ohmyz.sh/) and the plugins:

- git
- docker
- python
- zsh-syntax-highlighting
- zsh-autosuggestions
- ssh-agent
- web-search
